ramstart ramstart (tool command)

Acquires a starting logical block address (LBA) for a RAM area

Syntax

ramstart type

Description

Outputs the RAM area starting LBA for the disk of type "type". If the type value specified is less than 0 or greater than 6, an error occurs. In addition, since type-6 disks have no RAM area, a message to this effect is displayed if 6 is entered for "type".

For details, see Chapter 11 of the N64 Disk Drive Programming Manual.
