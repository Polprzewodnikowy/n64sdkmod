RDP Settings

<GBI commands>

gDPPipelineMode ... Turns coherent pipeline mode ON/OFF for the span cache buffer
gDPSetCycleType ... Sets the RDP cycle type.
gDPSetDepthSource ... Sets which source Z to use when comparing with the depth buffer
gDPSetColorImage ... Sets the color frame buffer area.
gDPSetDepthImage ... Sets the Z-buffer area.
