Cache Management

OS_DCACHE_ROUNDUP_ADDR
OS_DCACHE_ROUNDUP_SIZE		 ... Rounds up the values of the address and the size to match the data cache line size, and makes the address a physical address.
osInvalDCache ... Invalidates the CPU's data cache line.
osInvalICache ... Invalidates the CPU's instruction cache line.
osWritebackDCache
osWritebackDCacheAll		 ... Writes-back the CPU's data cache line to physical memory.
