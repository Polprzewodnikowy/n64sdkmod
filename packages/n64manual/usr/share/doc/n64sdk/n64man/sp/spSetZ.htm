spSetZ (function)

Sets the depth of a sprite.

Syntax

#include <sp.h> /* sp.h */
vvoid spSetZ(Sprite *sp, s32 z)

Arguments

*sp
Pointer to the sprite

Z
Z depth of a specified sprite

Description

It sets the depth of a specified sprite.

See Also

spDraw, spScale, spScissor
