Address Management (virtual & physical addresses)

OS_K0_TO_PHYSICAL ... Converts a cached, direct-mapped CPU virtual address (KSEG0) into a physical address.
OS_K1_TO_PHYSICAL ... Converts an uncached, direct-mapped CPU virtual address (KSEG1) into a physical address
OS_PHYSICAL_TO_K0 ... Converts a physical address into a cached, direct-mapped CPU virtual address (KSEG0).
OS_PHYSICAL_TO_K1 ... Converts a physical address into an uncached, direct-mapped CPU virtual address (KSEG1).
osVirtualToPhysical ... Converts a CPU virtual address into a physical address.
osPhysicalToVirtual ... Converts a CPU physical address into a virtual address.
osMapTLB ... Conducts CPU virtual address mapping.
osUnmapTLB ... Deletes CPU virtual address mapping.
osUnmapTLBAll ... Deletes all CPU virtual address mapping.
osSetTLBASID ... Sets the ID for CPU virtual address mapping.
osGetRegionBufSize ... Allocates arbitrary size of memory region.
