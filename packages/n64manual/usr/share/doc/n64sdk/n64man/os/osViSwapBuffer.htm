osViSwapBuffer [Function]

Function

osViSwapBuffer

Registers the frame buffer to be displayed in the next frame

Syntax

#include <ultra64.h>        /* os.h */
void osViSwapBuffer(void *vaddr);

Argument

vaddr
Head address of frame buffer

Return Value

None.

Description

This function registers the head address of the frame buffer to be displayed at the next vertical retrace interrupt.

See Also

osViGetCurrentFramebuffer, osViGetNextFramebuffer
