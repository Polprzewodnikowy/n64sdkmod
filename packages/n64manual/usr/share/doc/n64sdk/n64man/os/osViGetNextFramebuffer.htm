osViGetNextFramebuffer [Function]

Function

osViGetNextFramebuffer

Get the next frame buffer address

Syntax

#include <ultra64.h>        /* os.h */
void *osViGetNextFramebuffer(void);

Argument

None.

Return Value

Head address of frame buffer

Description

This function returns the head address of the frame buffer to be displayed next (Normally, the value set by the immediately preceding osViSwapBuffer is returned.)

See Also

osViGetCurrentMode, osViGetCurrentField, osViGetCurrentFramebuffer, and osViSwapBuffer
