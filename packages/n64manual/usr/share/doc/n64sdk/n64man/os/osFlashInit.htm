osFlashInit [Function]

Initializes PI handler for Flash.

Syntax

#include <ultra64.h>
OSPiHandle *osFlashInit(void);

Description

This is a function that initializes PI handler to use 1M Flash.

Execute this function once before using 1M Flash.  The return value is the pointer to PI handler for 1M Flash.  Normally, it is not necessary for a game application to use the return value of this function.
