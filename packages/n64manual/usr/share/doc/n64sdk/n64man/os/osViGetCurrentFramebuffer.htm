osViGetCurrentFramebuffer [Function]

Function

osViGetCurrentFramebuffer

Gets the current frame buffer address

Syntax

#include <ultra64.h>        /* os.h */
void *osViGetCurrentFramebuffer(void);

Argument

None.

Return Value

Head address of frame buffer

Description

This function returns the head address of the frame buffer currently being displayed.

See Also

osViGetCurrentField, osViGetNextFramebuffer
