gDPEndDisplayList

gDPEndDisplayList

Function

gDPEndDisplayList, gsDPEndDisplayList

Ends RDP display list

Syntax

#include "gt.h"
gDPEndDisplayList(Gfx *gdl)
gsDPEndDisplayList(void)

Arguments

*gdl
Pointer to the graphics display list

Description

This command ends an RDP command block in the turbo microcode.

This command does not function unless used with the gspTurbo3D.
