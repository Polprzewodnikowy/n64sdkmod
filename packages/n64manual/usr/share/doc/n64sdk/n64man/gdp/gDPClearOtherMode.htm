gDPClearOtherMode

gDPClearOtherMode

Function

gDPClearOtherMode, gsDPClearOtherMode

Initializes the RDP "othermode" command

Syntax

#include "gt.h"
gDPClearOtherMode(Gfx *gdl);
gsDPClearOtherMode(void);

Arguments

*gdl
Pointer to the graphics display list

Description

This command initializes the RDP "othermode" command with zeroes.

This command is usually used with gspTurbo3D micro code.

See Also

gspTurbo3D, gtStateSetOthermode
