Blender

<GBI commands>

gDPSetBlendColor ... Sets the blend color in the RDP.
gDPSetRenderMode ... Sets the rendering mode of the blender.
gDPSetAlphaCompare ... Sets the alpha compare mode for conditional writes to the frame buffer.
gDPSetAlphaDither ... Enables/disables alpha dithering.
gDPSetColorDither ... Enables/disables RGB color dithering.
gDPSetPrimDepth ... Sets the primitive depth (z) in the RDP.
