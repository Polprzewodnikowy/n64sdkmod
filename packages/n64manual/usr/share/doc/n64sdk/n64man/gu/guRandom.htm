Function

guRandom

Generates a pseudo-random number

Syntax

#include <ultra64.h>        /* gu.h */
int guRandom(void);

Arguments

None.

Returned value

32-bit integer (0 ~ 1073741823)

Description

Returns a pseudo-random 32-bit integer. You can specify an upper limit for the random number that is generated using RAND.
