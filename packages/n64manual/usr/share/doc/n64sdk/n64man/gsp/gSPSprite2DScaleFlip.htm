gSPSprite2DScaleFlip

gSPSprite2DScaleFlip

Function

gSPSprite2DScaleFlip, gsSPSprite2DScaleFlip

Specifies the basic sprite parameters.

Syntax

#include "gbi.h"
gSPSprite2DScaleFlip(Gfx *gdl, short TextureScaleX, short TextureScaleY, 
                        char FlipTextureX, char FlipTextureY);
gsSPSprite2DScaleFlip(         short TextureScaleX, short TextureScaleY, 
                        char FlipTextureX, char FlipTextureY);

Note

For more information about this command and its arguments, please refer to gspSprite2D.

See Also

gspSprite2D, gSPSprite2DBase, and gSPSprite2DDraw
