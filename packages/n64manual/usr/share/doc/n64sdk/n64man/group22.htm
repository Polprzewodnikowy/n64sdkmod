Controller

osContInit ... Detects and initializes the Controller.
osContReset ... Resets all Controllers.
osContSetCh ... Sets the number of Controllers to access.
osContStartQuery ... Executes a query command to read the Controller's status and type
osContGetQuery ... Gets the Controller's status and type.
osContStartReadData ... Executes a read command to get data from Controller.
osContGetReadData ... Gets data from Control Stick and Buttons.
