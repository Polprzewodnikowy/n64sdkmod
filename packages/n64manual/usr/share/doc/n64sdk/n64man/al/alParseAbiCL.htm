Function

alParseAbiC

Prints an ABI (audio binary interface) command list in human readable format

Syntax

#include "ultra64.h"
void alParseAbiCL(u64 *cmdList, u32 nbytes)

Arguments

cmdList
Pointer to the Audio Binary Interface (ABI) command list in DRAM

nbytes
Number of bytes in the ABI command list

Description

This function interprets and displays the commands in an ABI command list. For it to work correctly, you must be running the dlprint tool on the INDY host. The cmdList and nbytes arguments can be obtained by calling the alAudioFrame function. The cmdList argument is the same as the first argument to alAudioFrame, and nbytes is easily calculated by multiplying the returned value in the cmdLen parameter from alAudioFrame by sizeof(Acmd).

See Also

dlprint, guParseGbiDL, guParseRdpDL, and alAudioFrame
