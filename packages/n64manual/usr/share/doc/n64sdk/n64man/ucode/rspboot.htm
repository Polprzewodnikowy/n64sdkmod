rspboot

Function

rspboot

->RSP boot sequence microcode

Description

This small RSP microcode program enforces the OS synchronization between the host CPU (R4300) and the RSP. These are mostly software conventions, but they do include setting some hardware registers to begin a new task.

This microcode processes the task header structure, loads the microcode for the task, and begins the task execution.

The boot code does not remain in IMEM; it must be re-loaded at the beginning of each task.
