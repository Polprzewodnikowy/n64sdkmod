Controller Pak

osPfsAllocateFile ... Creates a new game note.
osPfsDeleteFile ... Deletes a game note.
osPfsFindFile ... Searches for the specified game note.
osPfsChecker ... Checks and repairs the Controller Pak's file system.
osPfsFileState ... Gets information concerning the specified game note.
osPfsNumFiles ... Gets the number of notes used in the Controller Pak.
osPfsFreeBlocks ... Get the size of free space (available number of bytes) in the Controller Pak.
osPfsIsPlug ... Detects the presence of a Controller Pak.
osPfsReSizeFile ... Adjusts size of note to specified length.
osPfsReadWriteFile ... Reads/writes data to the specified game note.
osPfsSetLabel ... Writes to the Controller Pak label.
osPfsGetLabel ... Reads the Controller Pak label.
osMotorInit ... Initializes the Rumble Pak
osMotorStart ... Starts the Rumble Pak.
osMotorStop ... Stops the Rumble Pak.
osPfsInitPak ... Initializes the Controller Pak's file handle.
osPfsRepairId ... Repairs the Controller Pak's ID region.
