bankdump (tool command)

Prints the symbol information contained in the N64 sound banks

Syntax

bankdump <symbol file>

Description

bankdump prints out the symbol information (symbol name, class, and offset) contained in the N64 sound banks into objects contained in the .sym file created by the ic (Instrument Compiler) tool.

The following is the command line options

<input file>
ic symbol file name
