sbc (tool command)

Makes a sequence bank file

Syntax

sbc [-o output_file] seqfile0 seqfile1 seqfile2 ...

Description

sbc takes a set of sequence files and collects them into a single sequence bank file, which is a convenient way of packaging multiple sequences for inclusion in the ROM.

Command Line Options

-o <filename>
specifies the sequence file that is written. The default output file is tst.sbk.
