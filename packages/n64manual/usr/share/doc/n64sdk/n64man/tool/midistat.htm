midistat (tool command)

Provides note statistics for MIDI files

Syntax

midistat [-v] <MIDI file>

Description

midistat is a tool that prints key numbers, the MIDI channel, and the number of notes per MIDI program number that are contained in the given sequence.

Command Line Options

-v
Turns on verbose mode.

<input file>
Is the name of a Type 1 or Type 0 Standard MIDI file.
