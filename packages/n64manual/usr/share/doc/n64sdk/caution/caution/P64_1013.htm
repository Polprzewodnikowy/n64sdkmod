10-13. Data Compatibility with Program Revisions (Required)

Description

Saved data in the Controller Pak must be compatible with any version (U.S., PAL, Japanese) of the same game, even when program modification occurs after commercial production has started.

Reason

The Controller Pak is a portable medium which can be carried by the user and exchanged between users. Data saved on a Controller Pak must be compatible with all versions of a particular game.
