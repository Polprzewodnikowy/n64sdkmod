10-6. Saving Game Notes (Required)

Description

Do not access a game note which has been produced by another company's game. However, the user should still be able to delete game notes which were created by another company's game.

Reason

This will prevent corruption of game notes created for other games. Accessing game notes should limited to the ones you have created.

Reference


Remarks

Please contact Nintendo first, if it is necessary to access game notes created by other parties.
