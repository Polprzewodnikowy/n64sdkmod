10-1. Compatibility of Data (Required)

Description

When sharing Game Pak data among countries, developers must use the same game code for all countries. Developers must not use Japanese characters or codes.

Reason

This is needed to ensure Controller Pak compatibility among countries.

Reference

Refer to "Nintendo 64 Developer News 1.3" "(4) Multi-National Compatibility of Controller Pak Data".

Remarks

English version of the libnos.a library for overseas use does not include Japanese font and displays Japanese characters as "-".
