14-1 Turning Off Power to Game Boy Game Pak (Required)

Description

Please use the function osGbpakPower() to turn off power to the Game Boy Game Pak when it is not being accessed.

Reason

It is possible that data may be lost when removing the Game Boy Game Pak while power is applied.

Please turn off power to the Game Boy Game Pak as soon as the communication process with the N64 Transfer Pak is completed, since the user might remove the Game Boy Game Pak during game play.

Reference

Please refer to the SI Device section of the N64 Programming Manual, "26.7 N64 Transfer Pak" for usage of the N64 Transfer Pak.
