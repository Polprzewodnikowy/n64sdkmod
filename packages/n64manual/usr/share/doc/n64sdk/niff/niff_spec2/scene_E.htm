4. SceneList

(1) SceneList

u32 scene_chain_root_link_num Describes the number of ChainRoot nodes directly linked from the scene.
u32 external_obj_num Describes the number of externally referenced Obj nodes.
u32 external_env_num Describes the number of externally referenced Env nodes.
u32 external_cam_num Describes the number of externally referenced Cam nodes.
u32 external_light_num Describes the number of externally referenced Light nodes.
u32 scene_chain_root_link_index Describes the index number for the ChainRoot nodes directly linked from the scene.

If scene_chain_root_link_num = 0, there is no data.

u32 external_obj_file_name_index Uses an ExternalNameList index number to specify the name of a file stored in the externally referenced Obj node.

Please do not include the directory name etc. when describing the file name.
