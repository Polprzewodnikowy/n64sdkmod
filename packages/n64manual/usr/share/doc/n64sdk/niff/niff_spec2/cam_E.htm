(1) CamNode

u32 external_lookat_obj_file_name_index When the Obj linked to the Cam look-at point references an external file, that file name is described with an ExternalNameList index number. In this case, please specify cam_lookat_obj = 0xffffffff.

Please do not include the directory name etc. when describing this file name.

If there is no external reference, please describe with 0xffffffff.

u32 external_lookat_obj_name_index Uses an ExternalNameList index number to describe the external file reference Obj name.

In this case, please specify cam_lookat_obj = 0xffffffff.

If there is no external reference, please describe with 0xffffffff.
