misRomInputEpiOpen

Opens MisRomInputEpi

Syntax

#include <mis/RomInputEpi.h>
void misRomInputEpiOpen( MisRomInputEpi *rom_input_ptr,
                         const u32 dev_addr,
                         const u32 size )

Description

Open MisRomInputEpi. Specify the first device address by dev_address and its size by size (a number of bytes).
