Function

mgiCharacterInit

Initializes character.

Syntax

#include <mgi/Character.h>
void mgiCharacterInit(   MgiCharacter *char_ptr,
                         MgiJobCallback callback,
                         void *user_data )

Description

Initializes character.
At the time of initialization, MgiControl still cannot be used.
