checknb

NIFF checker to check the NIFF1.0 specification parts

Syntax

checknb input-file

Explanation

Checks the NIFF data input-file. This command can handle both NIFF1.0 specification files and NIFF2.0 specification files, but it will not check the data newly established in NIFF2.0.

The NIFF data is read starting from the file header and output to standard output. In some instances the process will terminate on an error, but in other instances a Warning will be output and the read will continue.

Errors are output to standard error output, while Warnings are output as standard output following the data of each List.

What is checked is the size data and the tags. This command does not check whether the links are correct. No there checks are performed. For this reason, even if checknb and checknb2 do not detect any problems, this does not guarantee that the data is correct.

Notes

Since the data is read as much as possible even when errors occur, sometimes a "segmentation fault" error can arise.

There is no Windows version of this command.
