1. Revision History

Version Date Changes
2.02 1999/5/6 It is now possible to display the NIFF 2.0 CI Texture Type when using the function that displays texture data on Tex node.
2.01 1999/1/20 Settings for mat_type and mat_type_for_fog can be set separately using the Mat node.
2.00 1999/1/15 Support for NIFF 2.0
The startup script filename has been changed from "ne" to "ne2", and the NIFFeditor program filename has been changed from "niffeditor" to "niffeditor2". NIFF 1.0 files are compatible with versions of NIFFeditor up to and including version 1.35, and may be used with the old startup script ("ne") and program ("niffeditor").
New NIFF2tool menu has been added.
The number of displaying nodes has been increased to support the nodes added in NIFF 2.0.
The node setting dialogue box has been modified to support the nodes added in NIFF 2.0.
The N64 menu is disabled while reading a NIFF 2.0 file.
Internally, it was changed so that NIFFAPI is used. The change of NIFFAPI will be reflected.
A file can be opened while loaded data is deleted automatically, (the flow is now [File]-[Open]). Thus, the data does not have to be deleted manually in prior to open a file, (the previous flow was [file]-[New]-[Open}).
1.35 1998/10/19 The version of niff2gfx/n64prev became 1.3.5
1.34 1998/10/9 The version of niff2gfx/n64prev became 1.3.4
Resolved an issue where Quick Preview didn't always operate properly.
1.33 1998/10/1 The version of niff2gfx/n64prev became 1.3.3
1.32 1998/8/25 [Note] If using IS-VIEWER, please obtain the new version of nload that does not require makemask.
Support for Quick Preview If using IS-VIEWER, please obtain the new version of nload that does not require makemask.
niff2gfx/n64prev version changed to 1.3.2.
Supports the -noreduct option of niffgfx.
To support the new version of nload on IS-VIEWER, makemask is not performed during a the standard preview.
1.30 1998/7/15 Released with Developer Kit v3.0.
The editor no longer terminates when a file write error occurs.
Implemented niff2gfx/n64prev version 1.3.0.
The Preview Type setting is gone. Manipulations are now done on the Previewer.
1.20 1998/3/23 Registered for BBS and NTSC-ONLINE.
Added the menu (View->DisplayTextureImage) to view texture data on the Tex node.
The View->Default menu is now effective.
Support for changes in niff2gfx options (-vtxseg,-imgseg,-tlutseg,-strictcidx,-outmask,-preview)
It is now able to view LOD data.
Corrected problem that occurred when a link was added from Mat to Tex.
Object node LOD levels are now displayed under the LOD node and LOD values can be edited.
1.10 1997/12/24 Second Developer Kit released.
Editing is now possible for cam, env, light, anim, and partnodes.
Editing is now possible for name of scene, obj, shape, mat, and tex.
New links can now be created.
Made host-side transfers possible with obj-shape, obj-anim, obj-mat, shape-mat, part-mat, scene-obj, obj-obj, and mat-tex links.
Supported the increase in niff2gfx options. However, fname is not supported.
Added [Edit]-[Delete Link] menu. It is now possible to delete links.
Made impossible to change scene-cam, scene-env, and shape-part links.
Fixed a problem with other node settings being changed when numerous node dialogs are opened and some setting is changed.
Busy dialog will be shown up, when you take too much time to go preview with preview command.
A Warning Dialog now appears when an error occurs during preview.
The [Edit]-[EditFlags] menu has been eliminated.
1.01 1997/10/23 Registerd to BBS.
Fixed problem whit links "dropping" when double-clicked.
Designed the Select File dialog for SGI.
1.00 1997/10/9 Released first version with Developer Kit.
