Object Node Properties

The following tabs are available in the Object Node Properties dialog box:

  • Node name
  • State
  • Type
  • Group
  • Priority
  • Render
  • Render for Fog (NIFF 2.0j)
  • Billboard

Node name

The [Node name] tab sets the name of the Object node.

If the node is linked, the node name is displayed in the [Node name for object text box. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the name that is displayed. To specify a new name for a node that is not linked, delete "Unknown" and input the desired name. Conversely, to delete the node name, delete the name that is displayed in the text box. Please note that ASCII characters from "!" (0x21) to "~" (0x7e) can be used for node names. In addition, node names cannot contain spaces, nor can they contain kanji, hiragana or katakana characters.

State

Use the [State] tab to specify the state of the object.

Select either of the following for the [object state]:

  • [Ignore all links to the object]
  • [Activate the object]

Type

Use the [Type] tab to specify the object type.

Select one of the following types for the [object type]:

  • [NULL object]
  • [Object that has polygon data]
  • [Billboard-type object]

Group

Use the [Group] tab to specify the object group number.

When you want to divide an object into groups by type and assign group numbers, select [Set object group]. Then enter the group number as a hexadecimal value in the [Object group number] field.

Priority

Use the [Priority] tab to set priority order among objects.

To specify a priority order among objects of the same type, select [Set object's priority]. Then enter the priority order as a hexadecimal value in the [Object's priority] field.

Render

Use the [Render] tab to specify settings for rendering objects.

Under [Render cycle], choose one of the following cycle modes:

  • [1 cycle]
  • [2 cycle]

You may also choose one of the following render modes under [Render mode]:

  • [NIFF mode]
  • [N64GBI mode]

If you choose [NIFF mode], you can specify Render mode. If you choose [N64GBI mode], you can specify [Priority], [Render mode 0] and [Render mode 1].

Render for Fog

Use the [Render for Fog] tab to specify settings for rendering objects in a scene in which fog is used.

Under [Render cycle], choose one of the following cycle modes:

  • [1 cycle]
  • [2 cycle]

You may also choose one of the following render modes under [Render mode]:

  • [NIFF mode]
  • [N64GBI mode]

If you choose [NIFF mode], you can specify Render mode. If you choose [N64GBI mode], you can specify [Priority], [Render mode 0] and [Render mode 1].

Billboard

Use the [Billboard] tab to specify the presence or absence of billboard objects.

When a billboard object is included in the low-level links of the object, check the [Include hierarchy link to billboard object] box.
