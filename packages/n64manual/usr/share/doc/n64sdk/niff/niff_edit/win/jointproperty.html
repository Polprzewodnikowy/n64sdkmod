Joint Node Properties

The following tabs are available in the Joint Node Properties dialog box.

  • Node name
  • Type

Node name

Use the [Node Name] tab to specify the name of the Joint node.

If the node is linked, the node name is displayed in the [Node name for joint root] text box. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the name that is displayed. To specify a new name for a node that is not linked, delete "Unknown" and input the desired name. Conversely, to delete the node name, delete the name that is displayed in the text box. Please note that ASCII characters from "!" (0x21) to "~" (0x7e) can be used for node names. In addition, node names cannot contain spaces, nor can they contain kanji, hiragana or katakana characters.

Type

Use the [Type] tab to specify the joint type.

Choose one of the following Joint types under [Kind].

  • 1st joint
  • 2nd joint

Choose one of the following Joint directions under [Direction].

  • Up direction
  • Down direction
