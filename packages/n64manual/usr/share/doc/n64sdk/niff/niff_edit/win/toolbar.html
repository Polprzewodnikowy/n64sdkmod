Toolbar

Frequently used functions are assigned to buttons on the toolbar located at the top of the NIFF Editor window.

The function assigned to each button is explained below:

Create a new NIFF file
Open a NIFF file in NIFF Editor. When this button is selected the Open dialog box is displayed, enabling you to select a file from the list provided, or input the name of the NIFF file to open.
Saves the current contents of the open NIFF file by overwriting to the file with the same file name.
Cut a node and copy it to the clipboard.
Copy a node to the clipboard.
Paste the node that has been copied to the clipboard.
Show large icons in the list displayed on the right side of the NIFF Editor window.
Show small icons in the list displayed on the right side of the NIFF Editor window.
Display a simple list on the left side of the NIFF Editor window.
Include details about each item in the list displayed on the left side of the NIFF Editor window.
Convert the loaded NIFF file into a display list.
Send the loaded NIFF file to IS-VIEWER64 for preview.
Send the loaded NIFF file to IS-VIEWER64 for quick preview.
Convert the loaded NIFF file to an NVF file.
Send the loaded NIFF file to IS-Viewer64 to preview it.
Display NIFF Editor version information.

Show or hide the toolbar by toggling [toolbar] in the [View] menu.

By default, the toolbar is located just below the main menu. To reposition the toolbar anywhere on the screen, drag the button frame.
