Shape Node Properties

The following tabs are available in the Shape Node Properties dialog box.

  • Node name
  • Type
  • Fog (NIFF 2.0j)

Node name

The [Node name] tab sets the name of the Shape node.

If the node is linked, the node name is displayed in the [Node name for shape] text box. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the name that is displayed. To specify a new name for a node that is not linked, delete "Unknown" and input the desired name. Conversely, to delete the node name, delete the name that is displayed in the text box. Please note that ASCII characters from "!" (0x21) to "~" (0x7e) can be used for node names. In addition, node names cannot contain spaces, nor can they contain kanji, hiragana or katakana characters.

Type

Use the [Type] tab to specify the shape type.

Under [Alias], select one of the following alias processes:

  • [Perform anti-aliasing]
  • [Perform reduced aliasing]
  • [Clear the shape type]

Under [Z buffer], check the box to [perform Z-buffering] on the shape.

Under [Culling], specify whether you want to perform back-face culling or front-face culling by checking the appropriate box. To perform both back- and front-face culling, check both boxes. For no culling, leave both boxes empty.

Fog

Use the [Fog] tab to specify whether or not to use vertex alpha when using fog.

Select the "Use vertex alpha" box to use fog.
