FAQ

Below are some Frequently Asked Questions (FAQ) concerning the use of NIFF Editor.

How do I update to the latest version of NIFF Editor?

Support for NIFF Editor is available at the NOA Engineering website. You can download the latest version of NIFF Editor from the site.

I updated NIFF Editor but nothing seems to have changed.

If NIFF Editor has been previously installed, you must delete all of the files and sub-directories located in the following directory:

c:/nintendo/n64kit/niff/bin/niffedit   (this is the default location)

..prior to installing a new version of the software.

To delete NIFF Editor, from the Start button menu choose Settings, Control Panel, Add/Remove Programs. Follow the on screen instructions to delete NIFF Editor.

How do I change the home page address displayed in Online Support in the [Help] menu?

The Online Support home page address is set with the following Registry key:

HKEY_CURRENT_USER/Software/Intelligent Systems/NIFF Editor/Settings/OnlineAddress

Edit this key to specify a new address. Before doing so, carefully backup the existing Registry since changes can lead to problems with NIFF Editor and possibly Windows.

When I select [Quick Preview] from the [Tools] menu, I can't preview.

When Quick Preview is selected, NIFF Editor executes the quickprev.bat file. This batch file describes a process for IS-VIEWER64 to load the NIFF file without first having to perform makemask. This works because the makemask function is contained in nload.exe, for versions 1.02t and later. If you are using a version of nload.exe created prior to version 1.03, you won't be able to preview files with IS-VIEWER64 because makemask will not be performed.

If you are using a version of nload.exe created prior to version 1.03p, please add makemask process to the quickprev.bat file or download the latest version of nload.exe from the NOA Engineering website.

An application error occurs when click events or drag & drop events are performed on the right side window of NIFF Editor.

This error occurs when the version of comctl32.dll, Windows system file is 4.00.950. This error can be avoided by updating comctl32.dll in one of the following ways.

  • comctl32.dll is updated by installing Internet Explorer of version 3.0 or later. If Internet Explorer is uninstalled, comctl32.dll will not be removed.

  • Download the program to update comctl32.dll from Microsoft

Windows 95 later than OSR2, Windows 98, and Windows NT 4.0 contains comctl32.dll that doesn't produce this error.
