Animation Node Properties

The following tabs are available in the Animation Node Properties dialog box:

  • Node name
  • Type
  • Frame rate
  • Loop
Node name

The [Node name] tab sets the name of the Animation node.

If the node is linked, the node name is displayed in the [Node name for animation] text box. If it is not linked, then "Unknown" is displayed here. To change the node name, edit the name that is displayed. To specify a new name for a node that is not linked, delete "Unknown" and input the desired name. Conversely, to delete the node name, delete the name that is displayed in the text box. Please note that ASCII characters from "!" (0x21) to "~" (0x7e) can be used for node names. In addition, node names cannot contain spaces, nor can they contain kanji, hiragana or katakana characters.

Type

Use the [Type] tab to check the animation type setting.

The animation type setting cannot be changed.

Frame rate

Use the [Frame rate] tab to check the frame rate setting of the animation.

The animation frame rate setting cannot be changed.

Loop

Use the [Loop] tab to set the loop count for the animation.

To specify the number of animation loops, check the "Set number of animation loops" box. The loop number (for "Set number of animation loops") can be set using the decimal system.
