13.1 Graphics Binary Interface for Texture

The graphics binary interface (GBI) is a set of macros that create 64-bit commands that are read and parsed by the RSP microcode. Some of these commands cause actions or state changes in the RSP. Others are simply passed through the RSP to the RDP. Below is a list of GBI commands that control texture. See the corresponding reference page for more details.

13.1.1 Primitive Commands

  • g*SPTexture
  • g*SPTextureRectangle*

13.1.2 Tile Related Commands

  • g*DPSetTile
  • g*DPSetTileSize

13.1.3 Load Commands

  • g*DPLoadTile
  • g*DPLoadTextureBlock*, guLoadTextureBlockMipMap
  • g*DPLoadTLUT*
  • g*DPSetTextureImage

13.1.4 Sync Commands

  • g*DPLoadSync
  • g*DPTileSync

13.1.5 Mode Commands

  • g*DPSetTextureLUT
  • g*DPSetTexturePersp
  • g*DPSetTextureDetail
  • g*DPSetTextureLOD
  • g*DPSetTextureFilter
  • g*DPSetTextureConvert
