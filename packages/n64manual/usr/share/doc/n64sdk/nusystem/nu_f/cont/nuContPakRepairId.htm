Function

nuContPakRepairId

Syntax 

s32 nuContPakRepairId(NUContPakFile *file)

Arguments 

file      Controller Pak file structure

Return Value 

Error

Description

Repairs the Controller Pak file system

Checks the ID area of the Controller Pak and repairs the file system if it is corrupted. This is normally used when PFS_ERR_ID_FATAL is returned by the nuContPakOpen function.

Errors

PFS_ERR_NOPACK Nothing is inserted in the specified Controller.
PFS_ERR_NEW_PACK The Controller Pak may have been changed. Please perform the Open process again.
PFS_ERR_CONTRFAIL Data transfer with Controller has failed.
PFS_ERR_ID_FATAL Either the Controller Pak is damaged, it is not properly connected, or some peripheral without SRAM is inserted.
PFS_ERR_DVICE Some device other than a Controller Pak or a Rumble Pak is inserted.
