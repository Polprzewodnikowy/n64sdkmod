Function

nuAuSeqPlayerSetChlVol (Macro)

Syntax 

void nuAuSeqPlayerSetChlVol(u32 player_no, u8 chan, u8 vol)

Arguments 

player_no      sequence player number 
chan           MIDI channel number 
vol            volume value(0 - 127)

Return Value 

None

Description

Set volume for the MIDI channel specified by the player, which is specified by the sequence player number. The Volume controller (Cntrl7) has priority to affect this call.
