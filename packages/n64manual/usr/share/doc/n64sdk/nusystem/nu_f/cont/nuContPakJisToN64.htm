Function

nuContPakJisToN64

Syntax 

void nuContPakJisToN64(u8* src, u8* dest, u32 len)
Arguments 
*src      pointer to conversion source character string
*dest     pointer to conversion destination character string
len       length of conversion source character string
 Return Value 
None
 Description

Converts JIS code into N64 font code

Converts the JIS code specified by "src" into N64 font code and stores it in "dest." "len" is the length of the string specified by "src." Katakana daku-ten and handaku-ten are converted into 1 byte of corresponding N64 font code.
