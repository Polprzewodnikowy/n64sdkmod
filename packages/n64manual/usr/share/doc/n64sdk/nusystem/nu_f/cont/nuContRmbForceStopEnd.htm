Function

nuContRmbForceStopEnd

Syntax 

void nuContRmbForceStopEnd(void)
Arguments 
None
 Return Value 
None
 Description

End the forced stop on the Rumble Pak

Ends the forced stop on the Rumble Pak so the motor can be moved again. To force the Rumble Pak to stop, use the nuContRmbForceStop function.

See Also

nuContRmbStart and nuContRmbForceStop
