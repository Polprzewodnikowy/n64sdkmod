Function

nuAuStlSndPlayerSetSndVol (Macro)

Syntax 

s32 nuAuStlSndPlayerSetSndVol(u32 handle, s32 volume)

Arguments 

handle      sound handle
volume      volume scale ( 0 - 0x100 )

Return Value 

Number of channels for which command was executed

Description

Set the volume of the sound which is specified by the sound handle.
