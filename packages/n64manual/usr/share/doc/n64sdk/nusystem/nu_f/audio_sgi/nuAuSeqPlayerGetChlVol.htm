Function

nuAuSeqPlayerGetChlVol (Macro)

Syntax 

u8 nuAuSeqPlayerGetChlVol(u32 player_no, u8 chan)

Arguments 

player_no      sequence player number
chan           MIDI channel

Return Value 

Volume values (0 - 127) that correspond to the MIDI channel.

Description

Obtain volume for the MIDI channel specified by the player which is specified by sequence player number.

See Also

alCSPGetChlVol
