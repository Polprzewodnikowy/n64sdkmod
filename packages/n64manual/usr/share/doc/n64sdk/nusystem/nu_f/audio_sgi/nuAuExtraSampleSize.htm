Global Variable

nuAuExtraSampleSize

Definition 

s16 nuAuExtraSampleSize

Description

Extra sample size

Set the adjusted size of the audio data which are created at one time. This is normally 0 (zero).

If you want to change the value of this variable, change it before calling nuAuInit,nuAuInitEx, nuAuMgrInit, and nuAuMgrInitEx functions.
