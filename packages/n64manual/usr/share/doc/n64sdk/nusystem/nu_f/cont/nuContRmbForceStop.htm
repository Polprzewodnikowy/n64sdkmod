Function

nuContRmbForceStop

Syntax 

void nuContRmbForceStop(void)
Arguments 
None
 Return Value 
None
 Description

Forces the Rumble Pak to stop

When this function is called, the Rumble Pak Manager performs the process to forcibly stop the Rumble Pak. To remove the forced stop, use the nuContRmbForceStopEnd function. After the nuContRmbForceStop function has been called, the Rumble Pak motor will not move even if the nuContRmbStart function is called.

See Also

nuContRmbStart and nuContRmbForceStopEnd
