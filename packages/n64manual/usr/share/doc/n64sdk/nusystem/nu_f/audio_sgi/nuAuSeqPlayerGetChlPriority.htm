Function

nuAuSeqPlayerGetChlPriority (Macro)

Syntax 

u8 nuAuSeqPlayerGetChlPriority(u32 player_no, u8 chan)

Arguments 

player_no      sequence player number 
chan           MIDI channel

Return Value 

Priority(0-127)

Description

Obtain the priority order for the MIDI channel specified by the player which is specified by sequence player number. The lowest priority value is 0 and the highest priority value is 127.
