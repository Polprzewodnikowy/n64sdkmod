Global Variable

nuAuRetraceCount

Definition 

u8 nuAuRetraceCount

Description

Audio processing unit

This is the size of audio data created at one time. When 1 is specified, one frame of audio data is created. Normally, this is set to the same value as the retrace period set in the scheduler. The default value is 1.

If you want to change the value of this variable, change it before calling nuAuInit, nuAuInitEx, nuAuMgrInit, and nuAuMgrInitEx functions.
