Function

nuAuStlSeqPlayerGetSeqState (Macro)

Syntax 

s32 nuAuStlSeqPlayerGetSeqState(u32 player_no)
Arguments 
playr_no Sequence player number

Return Value 

Number of channels used

Description

Returns the number of channels used by the sequence specified by the sequence player number.

Caution

NuSystem version 2.00 and later is compatible with N64 Sound Tools version 3.xx. Please note that the specifications of functions have changed from prior versions.
