Function

nuAuStlSndPlayerDataSet

Syntax: 

void    nuAuStlSndPlayerDataSet(u8 *snd_addr, u32 snd_size)

Argument: 

snd_addr         ROM address for binary effect bank
ssnd_size        Size for binary effect bank

Return Value: 

None

Description:

Reads binary effect bank into the buffer of the sound player and initializes it.
