Function

nuAuSeqPlayerGetChlFXMix (Macro)

Syntax 

u8 nuAuSeqPlayerGetChlFXMix(u32 player_no, u8 chan)

Arguments 

player_no      sequence player number 
chan           MIDI channel

Return Value 

Effect mix value (0-127)

Description

Obtain the effect mix value for the MIDI channel specified by the player which is specified by sequence player number. The sound is completely dry when the value is 0 and the sound is completely wet when the value is 127.
