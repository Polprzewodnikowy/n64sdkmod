Function

nuAuSndPlayerSetSound (Macro)

Syntax 

void nuAuSndPlayerSetSound(ALSndId SndId)

Arguments 

SndId      ID for sound to be played back

Return Value 

None

Description

Specify the sound object for sound player. This is used when specifying the sound for which stop or effect will be applied by using the sound ID which is returned by nuAuSndPlayerPlay().
