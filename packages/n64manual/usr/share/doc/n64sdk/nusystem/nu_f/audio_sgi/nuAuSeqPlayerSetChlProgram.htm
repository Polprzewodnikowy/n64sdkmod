Function

nuAuSeqPlayerSetChlProgram (Macro)

Syntax 

void nuAuSeqPlayerSetChlProgram(u32 player_no, u8 chan, s32 prog)

Arguments 

player_no      sequence player number
chan           MIDI channel number of the object
prog           program number to be allocated

Return Value 

None

Description

Allocate a program number to the MIDI channel specified by the player which is specified by sequence player number.

See Also

alCSPSetChlProgram
