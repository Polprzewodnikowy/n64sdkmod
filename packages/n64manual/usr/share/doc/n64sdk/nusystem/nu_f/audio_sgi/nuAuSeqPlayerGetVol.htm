Function

nuAuSeqPlayerGetVol (Macro)

Syntax 

s16 nuAuSeqPlayerGetVol(u32 player_no)

Arguments 

player_no      sequence player number

Return Value 

Sequence volume (0-0x7fff)

Description

Return volume for player which is specified by sequence player number. 0x7fff indicates that the volume is full and 0 indicates it is off.
