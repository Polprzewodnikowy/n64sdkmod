Function

nuAuStlSeqPlayerPlay

Syntax 

musHandle nuAuStlSeqPlayerPlay(u32 player_no)

Arguments 

playr_no      sequence player number

Return Value 

Sequence Handler

Description

Plays back a sequence. The parameters of a sequence can be modified using the sequence handler of the return value.

Caution

NuSystem version 2.00 and later is compatible with N64 Sound Tools version 3.xx. Please note that the specifications of functions have changed from prior versions.
