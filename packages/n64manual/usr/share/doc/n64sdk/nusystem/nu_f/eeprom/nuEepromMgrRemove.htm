Function

nuEepromMgrRemove

Syntax 

void nuEepromMgrRemove(void)

Arguments 

None

Return Value 

None

Description

Removes the EEPROM Manager

Removes the EEPROM Manager from the SI Manager. To register the EEPROM Manager use the nuEepromMgrInit function.

See Also

nuSiMgrInit and nuEepromMgrInit
