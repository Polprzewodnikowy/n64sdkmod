Function

nuAuSeqPlayerSetNo

Syntax 

void nuAuSeqPlayerSetNo(u32 player_no ,u32 seq_no)

Arguments 

player_no      sequence player number 
seq_no         sequence number

Return Value 

None

Description

Set sequence number in the player which is specified by sequence player number. Read optional MIDI sequence data from MIDI sequence bank data to the buffer of sequence player using this function.
