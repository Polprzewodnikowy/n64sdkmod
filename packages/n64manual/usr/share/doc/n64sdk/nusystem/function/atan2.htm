Math Library Functions

atan2, atan2f

Syntax 

#include <math.h>
double atan2(double y, double x);
float  atan2f(float y, float x);
Arguments 
x
y

Return Value 

The arc tangent of y divided by x (y/x)

Description

Returns the arc tangent of y divided by x (y/x): 

atan(t) = t / (1+(t^2/(3+(2^2*t^2/(5+(3^2*t^2/(7+....)))))))
