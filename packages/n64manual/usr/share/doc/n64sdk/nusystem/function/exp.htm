Math Library Functions

exp, expf

Syntax 

#include <math.h>
double exp(double x);
float  expf(float x);

Arguments 

x      exponent

Return Value 

e to the power of x

Description

Calculates e to the power of x using the following formula: 

exp(x) = 1 + x + x^2/2! + x^3/3! + x^4 /4! +....... (-1<= x <=1)
exp(nk*log2+xx) = exp(nk * log2) * exp(xx)
		= 2^nk * exp(xx)
