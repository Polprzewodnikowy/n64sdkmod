3.4 IPL

The IPL is a program that starts a game from disk. It is stored in the mask ROM built into the 64DD. This program runs when power to the N64 is turned on without a Pak being inserted. The main functions of this program are:

  1. Checking expansion memory.
  2. Prompting the user to insert a disk.
  3. Displaying and setting the clock.
  4. Starting a game after a disk is inserted.
  5. Storing the ID of a loaded game in RDRAM.

For details on these tasks, please see "Chapter 4, 64DD Internal ROM."
