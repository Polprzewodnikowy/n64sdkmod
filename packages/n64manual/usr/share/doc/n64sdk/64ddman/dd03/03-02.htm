3.2 mild (PC only)

Tasks similar to those performed with makedisk in the previous section can be performed with the PC version of 64DD by using the -DD64 option with mild. With makedisk, however, symbols are referenced using descriptions such as the following: 

extern char _textureSegmentDiskStart[],_textureSegmentDiskEnd[]

With mild, the following descriptions are used. 

extern short _textureSegmentDiskStart,_textureSegmentDiskEnd
