11.4 System Area

The leo manager reserves tracks on the disk as a system area. This area contains system-related information, such as the disk type (see section 11.5 Disk Types (ROM and RAM Areas)) and the disk ID number (see Chapter 6: Disk ID).

The following is the information written to the system area.

Disk Types

This number determines the proportion of writable RAM area and read-only ROM area. For details, please see section 11.5, Disk Types(ROM and RAM Areas))

IPL Size and Load Addresss

When power to the 64DD is turned on, a demonstration is performed by the IPL, the program that starts games. If the disk is inserted at this point, several blocks are automatically read, beginning from the starting block, and stored at a certain address in RDRAM. The IPL size and the load address are the number of blocks to read initially and the location in RDRAM where the read programs and data are to be stored, respectively. When loading is finished, the IPL performs various initialization tasks, then jumps to this load address.

Disk ID

A number used to identify individual disks. For details, please see Chapter 6: Disk ID.
