11.2 Zones

As mentioned in the preceding section, each surface of the disk is divided into zones, with head 0 used for zones 0-7 and head 1 used for zones 1-8. Blocks within the same zone are the same size. Thus, either head 0 or head 1 would give the same size for a block in zone 1.

Thus, for example, Zone 0 has 134 tracks, or 268 blocks, and 19,720 bytes of data can be stored in each block. The capacity of Zone 0 is therefore 268 x 19,720 bytes, or 5,284,960 bytes.
