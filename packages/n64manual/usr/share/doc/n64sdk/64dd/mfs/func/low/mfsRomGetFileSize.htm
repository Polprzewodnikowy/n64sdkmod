mfsRomGetFileSize
mfsRamGetFileSize
Syntax 
s32 mfsRomGetFileSize(MfsDirID dirID)
s32 mfsRamGetFileSize(MfsDirID dirID)

Arguments

dirID   Directory ID of file

Return value

File size (a negative value is an error)

Function

Obtains file size

Description

This function obtains and returns the size of the file specified by dirID.

Main errors
MFS_ERR_NO Normal termination
MFS_ERR_NOTFOUND Specified directory not found
