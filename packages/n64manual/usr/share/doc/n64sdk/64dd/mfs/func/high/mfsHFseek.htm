mfsHFseek

Syntax 

s32 mfsHFseek(MfsFile handle, s32 offset, s32 fromwhere)

Arguments

handle   File handle
offset   Amount to move
fromwhere   Origin of move

Return value

Error

Function

Performs head seek

Description

This function moves the file position of the file specified by handle.

Also, it moves the read/write head to the file position. The new position is located at a distance of offset bytes from the position specified by fromwhere.

The following values can be specified for "fromwhere."

MFS_SEEK_SET 0 Beginning of file
MFS_SEEK_CUR 1 Current file pointer position
MFS_SEEK_END 2 End of file

Main errors

MFS_ERR_NO Normal termination
MFS_ERR_FATAL File open error
The file is not opened.
MFS_ERR_DEVICE Device error
An error occurred in an internally called Leo function. Since a Leo function error is stored in the variable mfsError, check this variable to perform processing.
MFS_ERR_HEADER Header error
The header area is corrupted. If it is the RAM area, try to repair it.
