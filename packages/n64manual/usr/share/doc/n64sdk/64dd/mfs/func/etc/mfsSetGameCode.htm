mfsSetGameCode

Sytnax

void mfsSetGameCode(u8* companyCode, u8* gameCode)

Arguments

*companyCode   Company code
*gameCode   Game code

Return value

None

Function

Sets company code and game code

Description

This function sets the company code and game code that are set in the MFS library. To set only the company code or only the game code, specify NULL for the buffer corresponding to the code that is not to be set.
