mfsRamResetVolumeRC
Syntax 
s32 mfsRamResetVolumeRC(void)

Arguments

None

Return value

Error

Function

Resets update counter

Description

This function sets the update counter value of the volume to 0. The update counter is incremented when file, directory, or disk information has been changed in some way. It is also incremented when volume information has been changed. Therefore, it can be used, for example, as a random number seed or to check whether or not the disk contents have been changed illegally.

Main errors

MFS_ERR_NO Normal termination
MFS_ERR_NOTMFSDISK Not an MFS-format disk
The disk ID's disk use is not 1, or the header is different.
MFS_ERR_VERSION Different version MFS-format disk
The version is an unsupported version.
MFS_ERR_PROTECT Write protected
The medium is volume protected.
