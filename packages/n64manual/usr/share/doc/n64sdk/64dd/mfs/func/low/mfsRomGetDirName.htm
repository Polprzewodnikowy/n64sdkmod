mfsRomGetDirName
mfsRamGetDirName
Syntax 
s32 mfsRomGetDirName(MfsDirID dirID, u8* name)
s32 mfsRamGetDirName(MfsDirID dirID, u8* name)

Arguments

dirID   Directory ID
*name   Directory name

Return value

Error

Function

Obtains directory name

Description

This function obtains the name of the directory specified by dirID.

Main errors

MFS_ERR_NO Normal termination
MFS_ERR_NOTFOUND Specified directory not found
