mfsStrWCmp

Sytnax

s32 mfsStrWCmp(const u8* src1, const u8* src2)

Arguments

s1
s2

Return value

Comparison result

Function

Compares character strings (version supporting wild cards)

Description

This function compares character string s1 with character string s2 and returns 0 if they are the same or 1 otherwise. Wild card characters (* or ?) can be included in character string s1. Wild card characters cannot be included in character string s2.
