mfsDiskIdCheck
Syntax 
s32 mfsDiskIdCheck(void)

Arguments

None
Return value
Error
Function
Reads and checks disk ID
Description

This function reads the disk ID and compares it with the previously read disk ID. If the disk has been exchanged, MFS_ERR_DISKCHANGE is returned. The disk ID that was read is stored in the global variable mfsDiskInfo structure. Please reference this global variable is necessary.
