mfsHFread

Syntax 

s32 mfsHFread(MfsFile handle, void* buf, s32 len)

Arguments

*handle is the file handle
*buf is the pointer to buffer for storing data
*len is the size of data to be read

Return value

Size of data that was read (a negative value is an error)

Function

Reads data

Description

This function reads data of the file specified by handle.

Main errors

MFS_ERR_FATAL File open error
The file is not opened.
MFS_ERR_DEVICE Device error
An error occurred in an internally called Leo function. Since a Leo function error is stored in the variable mfsError, check this variable to perform processing.
MFS_ERR_HEADER Header error
The header area is corrupted. If it is the RAM area, try to repair it.
