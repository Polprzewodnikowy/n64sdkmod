mfsRomGetDirListNext
mfsRamGetDirListNext
Syntax 
s32 mfsRomGetDirListNext(MfsDirList* dirList)
s32 mfsRamGetDirListNext(MfsDirList* dirList)

Arguments

dirList   MfsDirList structure buffer

Return value

Directory ID (a negative value is an error)

Function

Continues to obtain directory contents

Description

This function continues the search of mfsR*mGetDirListFirst. The same values that were specified in mfsR*mGetDirListFirst are specified for dirList.

Main errors

MFS_ERR_SEARCHEND Content not in directory
