mfsRomGetDiskType
mfsRamGetDiskType
Syntax 
s32 mfsRomGetDiskType(void)
s32 mfsRamGetDiskType(void)

Arguments

None

Return value

Disk type (a negative value is an error)

Function

Obtains disk type

Description

This function obtains the disk type.

Main errors

MFS_ERR_NOTMFSDISK Not an MFS-format disk
The disk ID's disk use is not 1, or the header is different.
MFS_ERR_VERSION Different version disk
