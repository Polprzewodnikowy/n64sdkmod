NUD Development System

[Summary]

The basic set for N64 Disk Drive development. (Set includes a development-use drive, a set of 5 disks, a double-slot adapter, an IPLROM Pak, a Memory Pak, and labels.) Use this set together with a debugger for N64 Disk Drive development work.

NUS Flash ROM Gang Writer

[Summary]

NUS Flash ROM Gang Writer writes data in up to 8 "NUS Flash ROM Paks" simultaneously for final debugging on the Nintendo 64 Control Deck. It is connected to the host machine (IBM compatible PC) via SCSI, and comes with control software (not including SCSI board).

NUS Development Cassette (Flash ROM Pak)

[Summary]

NUS Flash ROM Pak [Model #:NUS-16F32SB-128M/NUS-16F32SB-256M/NUS-16F32SB-512M] :128,256,512Mbit Flash Memory and 4kbit EEPROM & 256Kbit SRAM are installed. Erasing and writing can be done as you like, using Flash Gang Writer.

Double-slot Adapter

[Summary]

The official item code reads "NUS-DCC KIBAN ASSY". This adapter extends the Game Pak slot of the Nintendo 64 Control Deck into double slots for using an SRAM Pak together with a Flash ROM Pak that doesn't support SRAM. It also has CICs for both PAL and NTSC to help in development for PAL with Partner-N64. It is bundled with the NUD Development System.

256K SRAM Cassette

[Summary]

This 256K-SRAM cassette with built-in battery backup can be used to develop SRAM programs with IS-VIEWER and Flash ROM. Use the upper connector for Partner. For IS_VIEWER and Flash ROM, use this adapter with the Double-slot Adapter (above).

16K EEPROM

[Summary]

This is a 16K EEPROM for Nintendo 64 software backup, which can be implemented to Flash ROM Pak, Partner-N64, IS-VIEWER64, Emulator Board, etc.

CIC-NUS-6102 for NTSC

[Summary]

This CIC chip for NTSC (designed for Japan and U.S.) is factory-implemented in various tools.

CIC-NUS-7101 for PAL

[Summary]

This CIC chip is designed for PAL and required for developing PAL software. It is implemented in Flash ROM, Partner-N64, IS-VIEWER64, etc. A Nintendo 64 Control Deck designed for PAL needs to be used in this case.

Crystal Oscilliator for Modifying Emulator Board (PAL)

[Summary]

When using an Emulator Board and developing PAL software (for Europe), this crystal needs to be substituted on the Emulator Board.

Nintendo 64 Control Deck for PAL

[Summary]

This is a Nintendo 64 Control Deck designed for European PAL. Nintendo will decide which countries in Europe will be included for the specification (Note: the Control Deck specification would be identical except for the language used for manuals.).

Nintendo 64 Control Deck for MPAL

[Summary]

This is a Nintendo 64 Control Deck designed for Brazilian MPAL. Manufactured and purchased in Brazil, it is somewhat expensive.
