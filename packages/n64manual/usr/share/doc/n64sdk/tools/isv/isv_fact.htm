Nintendo IS-Viewer64 Facts & Features

With previewing software (not included) that supports the IS-Viewer64, you can use IS-Viewer64 to see how your game or 3D model will look on an N64 system. The IS-Viewer64 is a ROM emulator for the N64 system. It uses a SCSI interface to provide fast downloads of 1 Megabyte per second. Because it uses a SCSI interface, you can connect it to either a PC or a Silicon Graphics workstation. Graphics tool makers will be providing previewer support for the IS-Viewer64 so that you can use it to preview modeling and animation data in real time.

 Features

  • Verify N64 ROM images.
  • Download and verify a variety of information from the host computer. IS-Viewer64 puts together a library for debugging purposes and uses printf() to show you the results.
  • Preview graphics characters or scenes that you=re developing for instant feedback showing how they look on an N64 system. Then you=ll be ready to fine tune your graphical elements using your graphics tools, or demo them for others.
