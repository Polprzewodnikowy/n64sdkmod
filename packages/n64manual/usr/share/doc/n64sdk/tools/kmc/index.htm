Partner-N64NW

Partner-N64NW Specifications

[Summary]

A Nintendo 64 program debug system that is connected to a Silicon Graphics "Indy" workstation via ethernet. It is connected to the Indy via a network adaptor using a debugger Pak in a slot of a modified Nintendo 64 Control Deck. The kit includes Partner-N64 Debugger, Modified Nintendo 64 Control Deck, Network Adapter, and control software.

model11: Supports up to 32Mbyte of emulation ROM

Partner-N64PC

Partner-N64PC Specifications

[Summary]

A Nintendo 64 program debug system that is connected via the IAS bus of an IBM/PC compatible machine, the expansion slot of a PC9800 Series machine, or a notebook computer PCMCIA-Type II (option). The complete set consists of a Partner-N64PC Debugger Assembly, Modified Nintendo 64 Control Deck, a dedicated interface board (PCI), and control software.

model11: Supports up to 32Mbyte of emulation ROM
