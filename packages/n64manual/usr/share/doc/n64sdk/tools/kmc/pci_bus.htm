PCI Bus Interface Board for Partner-N64PC

Starting with software version 1.1 and beyond, Partner-N64PC provides support for the PCI interface board and for Windows NT. The PCI interface board is capable of transferring data at 4MB/s which is twice as fast than the old ISA interface board.

The PCI interface is a PCI card conforming to the PCI-SIG PCI 2.1 standard. There are no switches or jumpers for the user to set on the PCI interface board. The I/O address for the PCI interface board is automatically set by the Plug & Play function. The PCI interface board doesn’t use any interrupt.
