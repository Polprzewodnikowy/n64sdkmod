Partner-N64NW for Silicon Graphics Workstations

Partner-N64NW is a complete, window-oriented, network-
based N64 development and debugging environment for the
IRIX operating system and Silicon Graphics workstations
on an Ethernet network.


Partner-N64NW Benefits:

Better Games with Fewer Bugs
The Partner-N64NW debugging process makes it easier to find the bugs in your games. You'll have more time to focus on the design and features of your games.

Improved Productivity
With Partner-N64NW, you can automate routine portions of the testing process with macros. The powerful macro control structure uses a C-like control structure (If, For, While, Do, Break, and so on), so you can define new commands or combine commands and breakpoints. You can use macros to develop and run a professional automated debugging system to debug your games. The macro language makes it easy to develop reusable debugging suites.

Other powerful features in Partner-N64NW also help you improve your productivity. For example, using the On-the-Fly feature, you can enter certain commands while the game program is running. You can use this feature to view and change memory, IO, and register contents. You can also measure your game program's execution time in microseconds.

Use source-level debugging to debug Assembler code, debug N64 Disk Drive games, or support all video formats. With a single key, you can specify a breakpoint or execute a game program up to the cursor thus eliminating the need for symbolic debugging.

Easy Debugging in Windows
Debugging is easier using the following windows in
Partner-N64NW:

  • Code window to see source code or a combination of source and Assembler
  • Command window to enter commands and macros and to show results
  • Register window to see register contents
  • Stack window to see stack contents
  • Local window to see local variables
  • Watch window to see stored watch data
  • Backtrace window to see the backward trace of a C function
  • History window to see the contents of the History buffer.
  • Symbol Expansion Window to see expanded global symbols
  • Inspect window to inspect and display the contents of variables
  • Memo window to enter notes while you're debugging

Fast, Shared Network Access
Partner-N64NW attaches right to your network; it is not resident in nor connected directly to any one particular workstation, so several users can share it. For example, several graphic artists can share a centrally located Partner-N64NW system when using the Apreview mode@ of a modeling tool.

Support for NTSC, PAL, & M/PAL
Partner-N64NW comes with an NTSC Modified N64 Control Deck as the target device. You can easily switch to PAL or M/PAL development by simply switching to a PAL or M/PAL Modified N64 Control Deck (available separately from Nintendo) to achieve the correct video format.
