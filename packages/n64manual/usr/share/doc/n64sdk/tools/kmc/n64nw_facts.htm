Partner-N64NW Facts & Features

Partner-N64NW is a complete, window-oriented, network-based N647 development and debugging environment for the IRIX operating system and Silicon Graphics workstations on an Ethernet network.

Features

  • Use Partner-N64NW on any Silicon Graphics workstation including but not limited to Indy, Indigo, Indigo2 IMPACT, O2, OCTANE, and Onyx workstations.
  • Experience high-capacity 32 MB ROM emulation and memory support.
  • Enjoy a fast, clean network connection using TCP/IP sockets
  • Develop games for the N64 Disk Drive.
  • Set up to 15 software breakpoints.
  • Set the hardware breakpoint at any address and read/write status in memory to suspend the game program when the CPU accesses that spot.
  • Instruct your game program to break on illegal ROM area access.
  • Enter debugging commands while the game program is running.
  • Measure game program execution time in microseconds.
  • Use source-level debugging to debug in Assembler, to debug N64 Disk Drive games, or to support all video formats. With a single key, you can specify a breakpoint or run a game program up to the current cursor
    position.
  • Reference or change any variable in a format conforming to the data structure of that variable with a single key stroke.
  • Use the macro language to define new commands by combining commands.
  • View a saved history of commands in the History window. Use this to recall a command that you're using repeatedly so you don't have to retype it.
  • Use windows to navigate, control, and debug your game program. You get a Code window and Command window plus a Register, Stack, Local, Watch, Backtrace, History, Inspect, Symbol Expansion, and Memo
    windows.
  • Use the PROF (profile) command to calculate the time required to run each game program subroutine.
  • Automatically open an Inspect window to view or modify constant data by simply double-clicking the constant in the Source window. Constant data can be similarly viewed or modified in the Watch and Local
    windows.
  • Run commands in the Code window by pressing keys. For example, move the cursor next to a variable and then press Ctrl+W to register it in the Watch window. To view a variable at the current cursor position, press Ctrl+V. Run commands in the Command window by enter the command or by pressing keys. For example, press Shift+F2 to bring up the History window so you can rerun a command.

Software

  • Partner-N64NW Debugger

Hardware

  • Partner-N64NW Debugger Assembly (32MB model 11 Emulator)
  • Network Adapter
  • Modified N64 Control Deck (NTSC)
  • 40-pin cable
  • N64 AC power supply
  • Power supply for the Network Adapter
  • N64 Controller and Stereo Audio/Video cable

Requirements

  • Silicon Graphics platform workstation
  • IRIX Operating System
  • N64 Library Files and Debugger Version 2.0 G or above (debugger port function osWriteHost is not supported)
  • X11R4, X11R5, or X11R6 windowing
  • Ethernet (10 BASE-T or 10 BASE-2)
  • TV set for viewing the game
