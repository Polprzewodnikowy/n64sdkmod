Partner-N64PC for Personal Computers

Partner-N64PC is a complete, feature-rich, windows- based, game development and debugging environment for N64 game developers. The hardware attaches right to your PC and the software runs in Windows 95 or Windows NT and includes the exe GCC compiler customized for the N64.

Benefits:

Better Games with Fewer Bugs
The Partner-N64PC debugging process makes it easier to find the bugs in your games. You'll have more time to focus on the design and features of your games.

Improved Productivity
With Partner-N64PC, you can automate routine portions of the debugging process with macros. The powerful macro language uses a C-like control structure (If, For, While, Do, Break, and so on), so you can define new commands or combine commands with breakpoints. Using the macro language, you can develop and run a professional automated debugging system. The macro language makes it easy to develop reusable debugging suites.

Other powerful features in Partner-N64PC also help you improve your productivity. For example, using the On-the-Fly feature, you can enter certain commands while the game program is running. You can use this feature to view and change memory, I/O, and register contents. You can also measure game program execution times in microseconds.

Use source-level debugging to debug Assembler code, debug games developed for the N64 Disk Drive, or to support all video formats. With a single key, you can specify a breakpoint or execute a game program up to the cursor thus eliminating the need for symbolic debugging.

Easy Debugging in 12+ Windows

Debugging is easier using windows. With Partner-N64PC, you can use the following 12 windows:

  • Two Code windows to see source code or combined source and Assembler
  • Command window to enter commands and macros and to show results
  • Memory window to see memory data
  • Register window to see register contents
  • Stack window to see stack contents
  • Local window to see contents of local variables when a break occurs
  • Watch window to see stored watch data
  • Backtrace window to see the backward trace of a C function
  • Break window to see the currently set breakpoints
  • Inspect window to inspect and display the contents of variables
  • Memo window to enter notes while you're debugging
  • No New Equipment to Buy

You can develop, test, and debug your N64 games right there on your Windows 95 desktop.  Support for NTSC, PAL, & M/PAL.

Partner-N64PC comes with an NTSC Modified N64 Control Deck as the target device. You can easily switch to PAL or M/PAL development by simply switching to a PAL or M/PAL Modified N64 Control Deck (available separately from Nintendo) to achieve the correct video format.
