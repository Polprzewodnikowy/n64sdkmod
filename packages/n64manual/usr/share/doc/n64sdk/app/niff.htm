NIFF SAMPLE PROGRAMS

NIFF1.0

* dragon.niff

A detailed description can be found in n64kit/niff/sample/dragon.txt.

* bathroom.niff

A detailed description can be found in n64kit/niff/sample/bathroom.txt.

* LMS-lod.niff

This is an LOD sample. The object will be switched from L to M and to S. This is a sample for NIFF1 and cannot be viewed with n64prev. Be sure to use nvfview for viewing the sample.

NIFF2.0

* Bridge.niff

* OyajiGirl.niff

* Rings.niff

* ClusterBird.niff

* 3hedra.niff

Ambient color and diffuse color from NIFF 2.0 materials are used. For example, in the case of the center object, the ambient color is green and the diffuse color is red.

* hilight_donut.niff

This is a sample of specular highlight data. The luster of the object is expressed using the highlight texture.

* texanim_star.niff

This is a sample of texture animation.

* env_animal.niff

This is a sample using IK and Envelope. The shape changes according to the curved condition of the IK. If you wish to know the relationship between the positions of the Joint and the Effector, switch the ICON menu in the Previewer to ON.

* kirikobu.niff

This is a sample using IK and Envelope. This uses an Up Vector Constraint on the Joint to express more complex movement than env_animal.niff.
