Purpose

1. What is the MONEGI Multi Viewer 64

The MONEGI Multi Viewer 64 (hereinafter, MMV64) is a viewer for NINTENDO64 (hereinafter, N64) development equipment.
The MMV64 can be affiliated with a variety of files.

The following

  • MONEGI Smart Pack A,B,C
  • PARTNER-N64
  • IS-VIEWER64

can be used as target N64 devices.

2. Operating Environment

MMV64 will operate under the following environments.

  • OS

    • Windows 95/98/NT Ver. 4.0 or later

  • Hardware

    • CPU : Pentium 200MHz or faster is recommended
    Memory : 64MB or more is recommended
    N64 : MONEGI Smart Pack A,B,C or PARTNER-N64 (ISA) or IS-VIEWER64
          * The PCI version PARTNER-N64 cannot be used.
          * PARTNER-N64 cannot be selected under Windows NT.
