Prior to submitting ROM's and/or artwork, developers must request a game code from NCL using this form. Once the game code and print code are determined they will be sent to you by fax. The print code must appear on all relevant printed material for the product. The NUS game code must be entered in the "ROM Registration Data" (on the ROM) and on the "Nintendo 64 Software Specification Sheet".
