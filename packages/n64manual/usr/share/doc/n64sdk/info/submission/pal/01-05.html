Master ROM Image File

Data Storage

Use DOS formatted Iomega compatible Zip Disk(s), SGI TAR format DAT Tape, CD ROM formatted per ISO 9660, or 3.5 inch 128 Mbyte MO Disk(s) in MS-DOS Ver. 5 format.

Data Format

Data format must be binary with no compression.

Register the File Name

Write the file name using the following conventions.

Affix a Label

Affix a label to the Disk, CD ROM, or DAT Tape. As illustrated below, the label should include:

- Company Name
- Game Title
- Product Code
- File Name
- File Name (Game Code Rom Version)
- Submission Date
- Check Sum
